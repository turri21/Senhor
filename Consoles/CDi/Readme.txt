Instructions
------------

1. Copy the "MiSTer_CDi" file in /media/fat
2. For the CD-i core to function properly you must add the lines below to your MiSTer.ini file.

[CD-i]
main=MiSTer_CDi

3. Place cdi200.rom as boot0.rom in /media/fat/games/CD-i. 
4. Place zx405042p__cdi_slave_2.0__b43t__zzmk9213.mc68hc705c8a_withtestrom.7206 as boot1.rom next to it.
